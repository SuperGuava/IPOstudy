# 브레인스토밍 결과 보고서

## 주제
Anti-Gravity IPO/기업분석 대시보드 기반 신규 서비스 아이디어

## 토론 요약

5라운드에 걸쳐 3개 초기 아이디어가 1개 최종안으로 수렴되었다.

- **Round 1**: IPO 알리미, 비교 분석 리포트 생성기, 품질 점수 랭킹 3개 제안. Critic은 TAM 협소성($15~20K), 규제 리스크, 법적 리스크를 지적
- **Round 2**: 아이디어 1 폐기, 2+3 통합. Peer Finder(투자자용) + Readiness Score(기업용) 2트랙 제안. 건당 과금에서 월 멤버십으로 전환 합의
- **Round 3**: 메인 타겟을 리테일 개인투자자 1개로 집중. Pre-IPO Checklist는 TAM $6K로 축소. "스코어" 표현을 "밸류에이션 레인지"로 변경하여 법적 리스크 회피
- **Round 4**: Pre-IPO Checklist 완전 제거. KSIC + TF-IDF 텍스트 유사도 peer 매칭 보강. 백테스트 50건, 정확도 70%+ 목표 설정. 법률 자문 $1.5K 필수. 1년차 ARR $864 산정 (손익 -$792)
- **Round 5**: 서비스명 "IPO Insight" 확정. Top 3 인사이트 + 1줄 AI 요약 구조. 가격 $9/월 (론칭 $4.99), 연 $79.99. MVP 전 3단계 검증(법률 → 랜딩페이지 $100 → 백테스트) 합의. CAC $0 전략(SEO + DART RSS 자동 블로그)

**핵심 합의**: 리테일 개인투자자 단일 타겟, 법률 자문 필수, MVP 전 전환율 검증 필수, Pre-IPO Checklist 제외

**미합의**: 론칭 가격($9 vs $4.99), ARR $864 지속가능성, TF-IDF 정확도 실측치 부재

## 최종 선정 아이디어

**IPO Insight**

리테일 개인투자자(20~40대)를 위한 IPO 비교 분석 자동 생성 서비스. KSIC + TF-IDF 텍스트 유사도 기반으로 상장사 peer를 매칭하고, AI가 Top 3 인사이트 + 1줄 요약을 제공한다. 월 $9 멤버십(론칭 프로모션 $4.99), 연간 $79.99. 상장 후 1/3/6개월 퍼포먼스 알림으로 retention 확보. CAC $0 전략(SEO + DART RSS 자동 블로그)으로 MAU 416, 전환율 2%, 1년차 ARR $864 목표.

### 핵심 기능

```
[회사명] IPO 분석 (상장일: YYYY-MM-DD)

Top 3 인사이트
1. [재무 건전성] 영업이익률 15.2% (동종업계 평균 8.1% 대비 +87%)
2. [시장 차별성] 글로벌 시장 점유율 3년간 12%→18% 성장
3. [밸류에이션] 동종업계 평균 PER 대비 -12% (참고용, 투자추천 아님)

한 줄 요약: 수익성 양호하나 경쟁 심화 리스크 존재
```

### 기술 스택

- Frontend: Next.js + Vercel
- Backend: Python FastAPI + Supabase (PostgreSQL)
- AI: OpenAI GPT-4
- Data: DART Open API + KIND + KRX
- Peer 매칭: KSIC 5자리 + TF-IDF 사업보고서 텍스트 유사도
- 인프라 비용: $13/월

## 최종 판정

**조건부 승인**

- Q1(사용자 가치) ✅, Q3(시장 차별성) ✅ 통과
- Q2(기술 실현성) ⚠️, Q4(비즈니스 모델) ⚠️ 조건 충족 필요
- 3가지 사전 검증(법률/전환율/백테스트) 모두 통과 시 정식 승인

### 승인 조건

1. **법률 자문 통과**: 밸류에이션 레인지 표현의 자본시장법 저촉 여부 확인 ($1.5K)
2. **랜딩페이지 전환율 2%+**: $100 Google Ads로 실제 전환율 검증
3. **백테스트 정확도 70%+**: KSIC+TF-IDF peer 매칭 50건 실측

### Kill Criteria

- 법률 자문에서 사업 모델 수정 불가 판정
- 랜딩페이지 전환율 1% 미만
- 백테스트 정확도 60% 미만
- 3개월 운영 후 MAU 100명 미달
